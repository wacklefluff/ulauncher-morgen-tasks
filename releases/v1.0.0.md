# v1.0.0

**Date**: 2026-02-06

## Highlights

- List, search, and create Morgen tasks from Ulauncher (`mg`)
- Cache-first task listing (minimizes API points) + one-shot refresh (`mg refresh` / `mg !`)
- Natural language due parsing for task creation (`@today`, `@tomorrow`, `@friday`, ISO dates/times)
- Runtime logging to `extension/logs/runtime.log`

## Install

1. Ulauncher → Preferences → Extensions → “Add extension”
2. Paste the GitHub repo URL
3. Configure API key in extension preferences
